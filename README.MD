# Новостной сайт #

### Сайт с новостями созданый при помощи Flask  ###

### Записк: ###
1. Скопируйте репозиторий
2. Создайте виртуальное окружение
   - ```python -m venv venv```
3. Установите нужные библиотеки
   - ```pip install -r requirements.txt```
4. Запустите flask
   - ```python -m flask run```
